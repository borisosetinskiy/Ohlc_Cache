## 📊 Примеры использования

### Базовые операции
```java
// Вставка свечи
candleTimeSeriesCache.insert("AAPL", 300000L, timestamp, 
                            open, high, low, close, volume, tick);

// Чтение данных
Candle[] result = new Candle[1000];
int count = candleTimeSeriesCache.batchRead(result, "AAPL", 
                                           MarketType.STOCK, 300000L, 
                                           startTime, 0, 1000, 
                                           Direction.FORWARD, SearchMode.EXACT);
```

### Пакетная обработка
```java
// Пакетная вставка
candleTimeSeriesCache.batchInsert("AAPL", 300000L, candles, 0, candles.length);
```

## 🔧 Конфигурация

### Основные настройки
```yaml
cache:
  chronicle:
    time-series:
      base-path: "data/timeseries-chronicle"
  index:
    time-series:
      queue-size: 131072
      base-path: "data/timeseries-index"
```

### Системные свойства
```properties
jvm.resource.tracing=true
check.thread.safety=true
jvm.safepoint.enabled=false
```

## 📈 Производительность

### Ожидаемые характеристики
- **Запись:** 100,000+ свечей/сек
- **Чтение:** 1,000,000+ свечей/сек
- **Латентность:** <1 мс для одиночных операций
- **Память:** ~52 байта на свечу

### Оптимизации
- Off-heap хранение данных
- Flyweight паттерн для объектов
- Асинхронная индексация
- Бинарный поиск в чанках
